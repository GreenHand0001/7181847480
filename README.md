# 项目简介

基于SpringBoot+Vue的图书借阅系统，致力于提供便捷的图书借阅体验和高效的系统运营工具。本项目采用前后端分离的架构设计，集成了AI推荐算法和多维度数据大屏，全面覆盖图书借阅核心业务。

## 核心功能

- **登录/注册**：
  - 支持账号密码登录
  - 记住密码功能
  - 新用户信息填写
- **前台首页**：
  - 推荐图书展示
  - 功能入口集成
- **图书列表**：
  - 全量图书展示
  - 分类筛选
  - 关键词搜索
- **图书详情**：
  - 图书信息查看
  - 评分与评论浏览
- **图书榜单**：
  - 热门图书排行
  - 多维度榜单切换
- **个人信息**：
  - 个人基础信息查看与编辑
- **借阅确认**：
  - 借阅信息核对
  - 借阅申请提交
- **我的借阅**：
  - 当前与历史借阅记录查看与筛选
- **我的收藏**：
  - 收藏列表管理
  - 快捷借阅与取消收藏
- **我的评论**：
  - 自有评论查看、编辑与删除
- **数据大屏**：
  - 多维度数据可视化
  - 运营指标实时展示
- **协同推荐**：
  - 基于用户行为的个性化图书推荐算法

## 适用场景

本项目适用于图书馆、学校、企业等场景，旨在为用户提供便捷的图书借阅途径，同时为管理员提供系统化的运营管理工具。

## 技术特点

- 前后端分离架构设计
- 集成AI推荐算法
- 多维度数据可视化
- 全生命周期管理
- 实时运营指标展示
